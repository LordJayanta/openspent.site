# OpenSpent Landing Page 🌐

**Official Landing Page & Marketing Website for OpenSpent - A Modern & Privacy-Focused Personal Finance App**

![License](https://img.shields.io/badge/license-Apache--2.0-blue)
![Framework](https://img.shields.io/badge/framework-React%2BVite-blueviolet)
![Language](https://img.shields.io/badge/language-TypeScript-blue)

This repository contains the source code for the **OpenSpent landing page website**—the primary entry point for discovering, learning about, and downloading the OpenSpent personal finance application.

> 🪙 **About OpenSpent**  
> OpenSpent (formerly SpendWise) is a lightweight, open-source, and privacy-focused expense manager designed to give you full control over your financial data. It stores everything locally on your device—no cloud, no tracking, just your data.
>
> 📱 **Main App Repository**: [https://github.com/LordJayanta/openspent](https://github.com/LordJayanta/openspent)

---

## 🎯 Purpose

This website serves as the official landing page for the OpenSpent project, providing:

- 📖 Clear information about OpenSpent's features and benefits
- 📥 Download links and installation instructions
- 🔗 Quick access to the main repository and documentation
- ℹ️ Project roadmap, updates, and latest release information
- 💬 Community engagement and contribution guidelines

---

## 🛠 Tech Stack

- **Framework**: [React](https://react.dev/) with [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: CSS/CSS Modules
- **Build Tool**: Vite
- **Package Manager**: bun
